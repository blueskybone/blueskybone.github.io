NodeJS手动安装

发表于 2021-09-24 更新于 2022-07-13 分类于 教程

记录Nodejs安装

Nodejs版本太新，需要降级到老版本。有三种途径：

1.使用 nvm 工具来管理 Node.js。这是 Windows 版本。

2.下载NodeJS的安装版。最新版在NodeJS官网，老版本下载地址

3.下载压缩包手动配置。

nvm老出问题，总是要报错exit status 1，无奈卸载。下载msi安装，进去就提示系统已经安装了最新版本，然后直接退出安装了。所有NodeJS文件夹和环境变量都删干净了，不知道他怎么检测出来的……只能手动安装了。

步骤

1.下载win-X64.zip文件，解压到D盘，文件夹重命名为NodeJS。

2.添加D:\NodeJS和D:\NodeJS\node_modules\npm到系统环境变量PATH。

3.打cmd，输入node -v和npm -v测试版本。

4.安装express。

新版本的NodeJS集成了npm，不用再下载安装。

node.js和npm版本对应表

npm 更改版本命令

npm install -g npm@7.7.6

我的当前版本：node: ‘v15.14.0’, npm: ‘7.7.6’
