proxy配置：

不要使用powershell来配置proxy，使用cmd。

set http_proxy="http://localhost:8080"	#改成你的端口号
set https_proxy="https://localhost:8080"	#改成你的端口号

不设置代理，docker build的时候容易卡在pull环境阶段。
