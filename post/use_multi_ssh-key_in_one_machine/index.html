同一台机器管理多个git账号

2025/06/29

Categories: Manual

Windows

创建ssh密钥对

ssh-keygen -t rsa -f id_rsa_name #生成rsa类型的ssh密钥对名称为id_rsa_name，避免冲突自己换一个名称

修改.ssh下config文件

Host github_account_1	#起一个别名
  HostName github.com	#指定为github域名
  User git			    
  IdentityFile C:\Users\Godot\.ssh\id_rsa_name_1 #指定私钥文件
  
Host  github_account_2	
  HostName github.com	
  User git			    
  IdentityFile C:\Users\Godot\.ssh\id_rsa_name_2

在你的本地git仓库下指定使用账户的用户名和邮箱，设置一次即可。

git config user.name yourname
git config user.email your@email.address
